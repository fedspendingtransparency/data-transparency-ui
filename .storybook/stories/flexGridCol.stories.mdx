# Flex Grid Col

## Auto-layout columns

Utilize breakpoint-specific column classes for easy column sizing without an explicit numbered class.

### Equal-width

When no column widths are specified the `FlexGridCol` component will render equal width columns.

<Canvas>
    <Story name='Equal-width'>
        <FlexGridRow>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
        </FlexGridRow>
    </Story>
</Canvas>

### Setting one column width

Auto-layout for flex grid columns also means you can set the width of one column and have the sibling columns automatically resize around it.

<Canvas>
    <Story name='Setting one column width'>
        <FlexGridRow>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
            <FlexGridCol mobile={6}>
                <div>mobile:column-6</div>
            </FlexGridCol>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
        </FlexGridRow>
    </Story>
</Canvas>

### Variable width content

Set the column width (or any breakpoint size) to `"auto"` to size columns based on the natural width of their content.

<Canvas>
    <Story name='Variable width content'>
        <FlexGridRow>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
            <FlexGridCol width="auto">
                <div>column-auto</div>
            </FlexGridCol>
            <FlexGridCol>
                <div>column</div>
            </FlexGridCol>
        </FlexGridRow>
    </Story>
</Canvas>

## Responsive classes

### Same at all breakpoints

For columns that should maintain the same proportion at any viewport width, use `<FlexGridCol>` and `<FlexGridCol width={[1-12]}>`. Specify a numbered width property when you need a column of a specific width; otherwise, default to `<FlexGridCol>`.

`<FlexGridCol width={[1-12]}>` set a fixed width of [n] grid-columns in a 12-column grid.

<Canvas>

                <div>column</div>
            </FlexGridCol>

### Stacked columns at narrow widths

Columns are full-width until the narrowest breakpoint specified in a `mobile`, `tablet`, or `desktop` property. For instance, using `<FlexGridCol tablet="fill">`, you can create a basic grid system that starts out stacked before displaying as columns at the `tablet` breakpoint.

<Canvas>
    <Story name='Stacked columns at narrow widths'>

            </FlexGridCol>
            <FlexGridCol tablet="fill">
                <div>tablet:column</div>
            </FlexGridCol>
            <FlexGridCol tablet="fill">
                <div>tablet:column</div>
            </FlexGridCol>
        </FlexGridRow>
        <FlexGridRow>
            <FlexGridCol tablet={8}>
                <div>tablet:column-8</div>
            </FlexGridCol>
            <FlexGridCol tablet={4}>
                <div>tablet:column-4</div>

    </Story>
</Canvas>

## Alignment

### Column wrapping

Items wrap when the sum of the grid columns is more than 12.

<Canvas>
    <Story name='Column wrapping'>
        <FlexGridRow>
            <FlexGridCol width={8}>
                <div>column-8</div>
            </FlexGridCol>
            <FlexGridCol width={3}>
                <div>column-3</div>
            </FlexGridCol>
            <FlexGridCol width={5}>
                <div>column-5</div>
            </FlexGridCol>
        </FlexGridRow>
    </Story>
</Canvas>

## Reordering

### Order classes

Use `"order"` property for controlling the visual order of your content. These properties are responsive, so you can set the order by breakpoint (e.g., `<FlexGridCol tablet={{ span: 4, order: 'last'}}>`).

<Canvas>
    <Story name='Order classes'>
        <FlexGridRow>
            <FlexGridCol tablet={{ span: 4, order: 'last'}}>
                <div>first, but last</div>
            </FlexGridCol>
            <FlexGridCol>
                <div>second, but unordered</div>
            </FlexGridCol>
            <FlexGridCol tablet={{ span: 4, order: 'first'}}>
                <div>third, but first</div>
            </FlexGridCol>
        </FlexGridRow>
    </Story>
</Canvas>
