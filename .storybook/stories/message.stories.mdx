## Generic Message Props:
 - `className`: A way to customize message stlyling. Gets appended to `usda-message_` to create a _modifier_ class in BEM methodology.
 - `title`: The first (bolded) line of text.
 - `description`: The second line of text.
 - `icon`: An icon to display above the text.
