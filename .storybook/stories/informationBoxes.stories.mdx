## This is Designed to be a [Controlled Component](https://reactjs.org/docs/forms.html#controlled-components)
This means the component relies on its parent to pass the following props:
 - `boxes`: An array of box objects with their properties below
      - `title`: (string | element), the title to display.
      - `type`: (string, required), the unique React key for each box.
      - `subtitle`: string, the subtitle to display.
      - `amount`: number, the amount to display.
      - `isMonetary`: boolean, if true will format the amount as monetary.
      - `isLoading`: element, if true will display loading vizualisation.

In this example:
